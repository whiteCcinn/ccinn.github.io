##关于 Ascii 码的整理

```
- 0 < ascii < 128 单字节字符 == 1xxx xxxx

向右移动 7 位等于 0.符合范围内

- 240 <= ascii 四字节字符 == 1111 xxxx

向右移动 4 位=15，符合范围内

- 224 <= ascii < 240 3 字节字符 == 111x xxxx

向右移动 5 位=7，符合范围内

- 192 <= ascii < 224 2 字节字符 11xx xxxx

向右移动 6 位=3，符合范围内
```
